# Projects

Things I've built and actually run. Mostly self-hosted, mostly Go/Elixir, all scratching a real itch.

## Running services

* [Werdz](https://werdz.ca) — multiplayer definition-bluffing word game. Fool your friends, score points.
* [Mermaidarium](https://mermaidarium.com) — render Mermaid diagrams to images, with metadata baked in so you can re-edit them later.
* [Eventshare](https://eventshare.app) — privacy-respecting calendar event sharing. Share an invite without feeding a data broker.
* [Quote Hole](https://quotes.onewheelgeek.net) — collect and vote on the best quotes from your crew.

## Tools & infrastructure

* [PullPilot](https://github.com/jclement/pullpilot) — secure, compose-aware Docker auto-updater with soak windows and health-gated rollback. It keeps this very server fresh.
* [Gatecrash](https://github.com/jclement/gatecrash) — self-hosted tunnel server; my answer to "do I really need Cloudflare for this?"
* [DrillBit](https://github.com/jclement/drillbit) — stop SSH-ing into machines just to poke at your databases.
* [Megaclaude](https://github.com/jclement/megaclaude) — sandboxed Claude Code in Docker, for letting an AI loose without letting it loose.
* [capsule](https://github.com/jclement/owg-fyi) — the Gemini + web server rendering this page right now. One Go binary, two protocols, zero JavaScript.
